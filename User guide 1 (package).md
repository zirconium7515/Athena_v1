Athena v1 사용 가이드 (1. 설치 및 실행편)이 가이드는 Athena v1 자동매매 프로그램을 설치하고 실행하는 방법을 안내합니다.1. 최소 요구 사항Python: 3.10 이상Node.js: 18.x 이상 (GUI 실행용)Git: (코드 다운로드 및 버전 관리용)2. 설치 및 설정프로그램 실행 전, PC에 필요한 환경을 설정합니다. (최초 1회)(권장) 1단계: Python 가상환경 구축터미널(PowerShell 또는 cmd)을 열고, Athena_v1 프로젝트 폴더에서 다음 명령어를 실행하여 가상환경(venv)을 생성하고 활성화합니다.# (위치: Athena_v1)

# 1. venv 라는 이름의 가상환경 폴더 생성
python -m venv venv

# 2. 가상환경 활성화 (Windows)
.\venv\Scripts\activate

# (참고) 가상환경 활성화 (macOS/Linux)
# source venv/bin/activate
가상환경이 활성화되면, 터미널 프롬프트 앞에 (venv)가 표시됩니다.2단계: 백엔드 (Python) 의존성 설치(가상환경이 활성화된 상태에서) requirements.txt 파일에 명시된 Python 라이브러리들을 설치합니다.# (위치: Athena_v1)
# (venv)가 활성화된 것을 확인

pip install -r requirements.txt
3단계: 프론트엔드 (React) 의존성 설치GUI를 실행하기 위해, 별도의 새 터미널을 열고 frontend 폴더로 이동하여 npm install을 실행합니다.# (위치: 새 터미널)

# 1. frontend 폴더로 이동
cd frontend

# 2. package.json에 명시된 라이브러리 설치 (node_modules 폴더 생성)
npm install
(참고: 이 작업은 node_modules라는 용량이 큰 폴더를 생성하며, 몇 분 정도 소요될 수 있습니다.)3. 프로그램 실행Athena v1은 **백엔드(서버)**와 프론트엔드(GUI) 두 개가 동시에 실행되어야 합니다. 두 개의 터미널이 필요합니다.터미널 1: 백엔드 (FastAPI) 실행(가상환경이 활성화된) 첫 번째 터미널이 Athena_v1 루트 폴더에 있는지 확인합니다.다음 명령어를 실행하여 백엔드 서버를 시작합니다.# (위치: Athena_v1)
# (venv)가 활성화된 것을 확인

uvicorn main:app --host 127.0.0.1 --port 8000 --reload
--reload 옵션: 소스 코드(.py)가 변경될 때마다 서버를 자동으로 재시작합니다.INFO: Uvicorn running on http://127.0.0.1:8000 메시지가 보이면 성공입니다.터미널 2: 프론트엔드 (React) 실행(npm install을 실행했던) 두 번째 터미널이 Athena_v1/frontend 폴더에 있는지 확인합니다.다음 명령어를 실행하여 React 개발 서버(GUI)를 시작합니다.# (위치: Athena_v1/frontend)

npm start
이 명령어를 실행하면, http://localhost:3000 주소로 브라우저가 자동으로 열립니다.4. GUI 사용법브라우저에 http://localhost:3000 주소로 접속하면 Athena v1 GUI가 나타납니다.0단계: API 키 설정 (필수)봇을 실행하기 전, 반드시 업비트에서 발급받은 API 키를 입력하고 저장해야 합니다.0. API 키 설정 섹션에 Access Key와 Secret Key를 입력합니다.[API 키 저장] 버튼을 누릅니다.백엔드 서버가 키를 인증합니다.(성공 시): "API 키 저장 및 인증 성공. (보유 KRW: ...)" 메시지가 표시됩니다.(실패 시): "API 키 인증 실패" 오류 메시지가 표시됩니다. (키가 정확한지, IP 주소가 등록되었는지 확인하세요.)1단계: 거래 코인 선택1. 거래 코인 선택 탭에서 자동매매를 실행할 코인을 선택합니다.검색창을 이용해 원하는 코인을 찾을 수 있습니다.코인을 클릭하면 선택(하늘색 배경)되거나 선택 해제됩니다.2단계: 봇 제어2. 봇 제어 탭의 버튼을 눌러 봇을 시작하거나 중지합니다.[선택 봇 시작]: 현재 1단계에서 선택된 코인들 중, (아직 실행 중이지 않은) 봇을 시작시킵니다.[선택 봇 중지]: 현재 1단계에서 선택된 코인들 중, (이미 실행 중인) 봇을 중지시킵니다.(참고: 코인 목록의 (실행중) 표시는 GUI가 아닌 백엔드 main.py의 active_bots 상태를 기준으로 합니다.)3단계: 실시간 로그 확인3. 실시간 로그 탭에서 백엔드 서버의 활동과 봇의 매매 신호, 주문 실행 내역을 실시간으로 확인할 수 있습니다.(error): 심각한 오류 (빨간색)(success): 봇 시작, 진입 성공 (초록색)(info): 일반 정보 (검은색)5. 프로그램 종료(터미널 2: React) Ctrl + C 키를 눌러 프론트엔드 서버를 종료합니다.(터미널 1: FastAPI) Ctrl + C 키를 눌러 백엔드 서버를 종료합니다.

Athena v1 - 새 컴퓨터 (초기화) 설정 가이드이 문서는 포맷된 새 Windows PC에서 Athena_v1 자동매매 프로그램을 실행하기 위해 필요한 모든 소프트웨어의 설치 및 설정 단계를 안내합니다.1. 필수 설치 프로그램 (총 3개)가장 먼저, 코드 편집기, 파이썬(벡엔드), Node.js(프론트엔드)를 설치해야 합니다.1-1. Visual Studio Code (코드 편집기 + 터미널)코드를 확인하고, 벡엔드/프론트엔드 실행 명령어를 입력할 터미널(명령 프롬프트)로 사용합니다.다운로드: https://code.visualstudio.com/설치: 다운로드한 파일을 실행하고, 설치 중 'PATH에 추가(Add to PATH)' 옵션이 선택되어 있는지 확인합니다.1-2. Python (벡엔드 실행용)main.py 등 벡엔드 서버를 실행하기 위해 필요합니다.다운로드: https://www.python.org/downloads/(권장: Python 3.10 또는 3.11 버전)설치 (매우 중요): 설치 프로그램(Installer) 실행 시, 첫 화면 하단의 "Add Python 3.xx to PATH" 체크박스를 반드시 클릭(체크)한 후 'Install Now'를 누릅니다. (이것을 체크하지 않으면 2단계에서 pip 명령어가 작동하지 않습니다.)1-3. Node.js (프론트엔드 실행용)App.js 등 프론트엔드(GUI) 화면을 실행하기 위해 필요합니다. npm이 함께 설치됩니다.다운로드: https://nodejs.org/(권장: LTS 버전)설치: 다운로드한 파일을 실행하고, 기본 옵션으로 설치를 완료합니다.2. 프로젝트 실행 단계 (벡엔드 + 프론트엔드)필수 프로그램 3개 설치가 완료되고, Athena_v1 프로젝트 폴더를 새 컴퓨터로 복사했다고 가정한 후의 단계입니다.VS Code를 실행합니다.상단 메뉴에서 File > Open Folder...를 클릭하여, 복사해 둔 Athena_v1 폴더 자체를 엽니다.VS Code 상단 메뉴에서 Terminal > New Terminal을 클릭하여, 화면 하단에 터미널(명령창)을 엽니다.2-1. 벡엔드 (Python) 설정 및 실행(권장: 가상 환경 생성) 터미널에 다음 명령어를 입력합니다.python -m venv venv
(권장: 가상 환경 활성화) 터미널에 다음 명령어를 입력합니다. (프롬프트 앞에 (venv)가 표시됩니다.).\venv\Scripts\activate
(필수) 파이썬 라이브러리 설치: Athena_v1 폴더에 requirements.txt 파일이 있는지 확인하고, 터미널에 다음 명령어를 입력합니다.pip install -r requirements.txt
벡엔드 서버 실행: 모든 라이브러리가 설치되면, 터미널에 다음 명령어를 입력합니다.uvicorn main:app --reload
확인: 터미널에 "Uvicorn running on https://www.google.com/search?q=http://127.0.0.1:8000" 메시지가 표시되면 벡엔드 실행 성공입니다.2-2. 프론트엔드 (React) 설정 및 실행벡엔드 서버가 실행 중인 터미널 옆의 + (플러스) 아이콘을 클릭하여, 두 번째 새 터미널을 엽니다.프론트엔드 폴더로 이동: 두 번째 터미널에 다음 명령어를 입력합니다.cd frontend
(필수) React 라이브러리 설치: 터미널에 다음 명령어를 입력합니다. (시간이 다소 걸릴 수 있습니다.)npm install
프론트엔드 서버 실행: 설치가 완료되면, 터미널에 다음 명령어를 입력합니다.npm start
확인: 잠시 후 "Compiled successfully" 메시지와 함께, 자동으로 새 브라우저 창이 열리며 http://localhost:3000 주소로 접속됩니다.모든 단계가 완료되었습니다. 이제 브라우저에서 프로그램을 사용할 수 있습니다.
